# hr-analytics-attrition-prediction

## Project Objective
This project analyzes employee attrition data to identify the main reasons employees leave the
company and predict future attrition using machine learning models.--

## Tools & Technologies : Python- Pandas- Seaborn- Scikit-learn- SHAP- Power BI-

## Project Workflow

### 1. Data Cleaning & EDA
Performed Exploratory Data Analysis (EDA) on HR data:- Department-wise attrition- Salary band analysis- Promotion impact- Overtime impact

### 2. Machine Learning Models
Built classification models:- Logistic Regression- Decision Tree Classifier

### 3. Model Evaluation
Evaluated model performance using:- Accuracy Score- Confusion Matrix- Classification Report

### 4. SHAP Analysis
Used SHAP values to explain feature importance and model predictions.

### 5. Power BI Dashboard
Created an interactive dashboard including:- Attrition Rate KPI- Department-wise Attrition- Overtime Impact- Promotion Impact- Salary Band Analysis- Interactive Slicers-

## Key Insights
- Employees working overtime have higher attrition.- Low salary employees are more likely to resign.- Fewer promotions increase attrition risk.- Sales department shows higher employee turnover.- Overtime and      salary are major factors affecting attrition.
